# Exercices Java : Incrémentation, Tableaux et Algorithmes

## Description 

Ce projet contient une série d’exercices en Java portant sur :
- L’incrémentation (pré-incrémentation et post-incrémentation)
- Les tableaux à une dimension
- Les algorithmes de tri
- Les algorithmes de recherche

## Contenu ( se trouvant dans le dossier code)

### Partie 1 : Incrémentation
- Différence entre `i++` et `++i`
- Expressions complexes avec incrémentation
- Réécriture sans opérateur `++`

### Partie 2 : Tableaux (1D)
- Déclaration et initialisation d’un tableau
- Parcours et affichage des éléments
- Calcul de la somme
- Recherche du maximum et du minimum
- Copie d’un tableau

### Partie 3 : Tri et Recherche
- Tri par sélection (Selection Sort)
- Tri à bulles (Bubble Sort)
- Principe du tri par insertion
- Recherche linéaire
- Recherche binaire (sur tableau trié)
